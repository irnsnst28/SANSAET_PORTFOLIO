# Aaron Sansaet Portfolio

A responsive React and TypeScript portfolio for Aaron Sansaet, focused on web
development, full-stack systems, UI/UX work, and video editing.

## Run locally

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

Portfolio content is maintained in `src/constants/portfolio.ts`. Contact-form
delivery uses EmailJS and the variables documented in `.env.example`; without
them, inquiries are retained locally in the browser.
